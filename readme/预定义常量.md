| 常量名         | 功能                 |
|-------------|--------------------|
| \_\_FIFE__  | 默认常量，PHP文件名        |
| \_\_LINE__  | 默认常量，PHP程序行数       |
| PHP_VERSION | 内建常量，PHP版本         |
| PHP_OS      | 内建常量，执行PHP解释器的操作系统 |
| TRUE        | 该常量是一个真值           |
| FALSE       | 该常量是一个假值           |
| NULL        | 一个NULL值            |
| E_ERROR     | 指到最近的错误处           |
| E_WARNING   | 指到最近的警告处           |
| E_PARSE     | 指到解析语法有潜在问题处       |
| E_NOTICE    | 指到不寻常但不一定错误处       |
